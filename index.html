Spielregeln

  • Spielziel: Mindestens 3 Steine der gleichen Farbe in eine Reihe bringen.
  • Spielablauf: Einen Stein auswählen
    1. Der Spieler wählt einen Stein.
    2. Dieser Stein kann durch klicken auf ein Nachbarfeld mit diesem getauscht werden.
    3. Wenn durch den Tausch eine Reihe von drei oder mehreren Steinen der gleichen Farbe entsteht, ist der Zug gültig und die Reihe wird gelöscht.
  • Punktesystem:
    • 3er Combo = 450Pkt
    • 4er Combo = 800Pkt
    • 5er Combo = 1250Pkt
    • ...
  • Das Spiel dauert 90 Sekunden.
